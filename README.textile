h2. INSTALLATION

<pre>
<code>git clone git://github.com/BetterCode/java-procedure-api.git
cd java-procedure-api
mvn install</code>
</pre>

h2. CONFIGURATION

Just put this in your pom.xml:

<pre>
<code>
<dependency>
	<groupId>bettercode</groupId>
	<artifactId>java-procedure-api</artifactId>
	<version>0.0.1-SNAPSHOT</version>
</dependency>
</code>
</pre>
